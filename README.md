## Disclaimer
This is an educational fan-made project.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/ivanodsp/Ivano-s-Tetris.git
   ```
2. Navigate to the project directory and start a local Python HTTP server:
   ```bash
   python -m http.server
   ```

## Features
- 🎮 Engaging gameplay
- 🌟 Stunning graphics
- 🎶 Sound effects

## How to Play
Use the arrow keys to move the Tetris pieces. Clear lines to earn points!
